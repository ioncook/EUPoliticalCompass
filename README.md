  To use, simply download the zip and extract it. Run the html and select the CSV in the package or supply another in the CHES-EU format. Alternatively, create a python server in the folder and it will automatically select the CSV when the page is loaded. 
  
  Listed "groups" do not directly correlate to real EU Parliament groups but fit loosely, the groups in the CSV coorespond mainly to party platform, which carries over groups. Names and descriptions can be easily edited in the HTML.
  
  Individual parties can be selected by clicking on their respective dot, groups by clicking on the group labels below the graph. Dropdown menus can be scrolled on to quickly change selections. The site is fully functional on mobile devices, although zoom may be required to interact with certain UI elements.
